# Timeweb WABA Landing

Статический одностраничный сайт на базе текущего лендинга Raznos, подготовленный для отдельного GitHub-репозитория и публикации через Timeweb App Platform.

## Что внутри

- `index.html` — главная страница сервиса с описанием Raznos и контактами
- `privacy.html` — политика конфиденциальности
- `styles.css` — стили для обеих страниц
