# Timeweb WABA Landing

Статический одностраничный сайт на базе текущего лендинга Raznos, подготовленный для отдельного GitHub-репозитория и публикации через Timeweb App Platform.

## Что внутри

- `index.html` — главная страница сервиса с описанием Raznos и контактами
- `privacy.html` — политика конфиденциальности
- `styles.css` — стили для обеих страниц

## Что нужно заменить перед публикацией

На обеих страницах сейчас указан временный email:

- `hello@your-domain.kz`

Перед публикацией замените его на реальный корпоративный email на вашем домене.

## Как вынести в GitHub

1. Создайте новый репозиторий, например `raznos-waba-site`.
2. Скопируйте содержимое этой папки в корень нового репозитория.
3. Загрузите репозиторий в GitHub.
4. Подключите репозиторий в Timeweb App Platform.
5. Привяжите домен и включите HTTPS.
